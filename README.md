 🎥 Nexalor

Nexalor est une plateforme de partage de vidéos intuitive, rapide et interactive, un peu comme un YouTube simplifié ! 🚀

---

## ✨ Fonctionnalités principales
* **Visionnage de vidéos :** Regarde tes contenus préférés de manière fluide.
* **Système de Likes :** Ajoute des likes sur les vidéos pour soutenir les créateurs.
* **Partage :** Une interface pensée pour mettre en ligne et parcourir du contenu facilement.

## 🛠️ Technologies utilisées
* HTML5 : Pour toute la structure logique et le squelette du site.
* CSS3 : Pour le design moderne, le style et l'interface (responsive).
* JavaScrip : Pour gérer le lecteur vidéo, les clics sur les boutons et les interactions comme les likes.

## 🚀 Comment lancer le projet ?
Pour tester le site sur ton ordinateur, c'est super simple :

1. Télécharge ce projet en cliquant sur le bouton vert **Code** puis **Download ZIP** (ou clone-le si tu connais Git).
2. Décompresse le fichier `.zip` sur ton PC.
3. Fais un double-clic sur le fichier **`index.html`** pour l'ouvrir directement dans ton navigateur web (Chrome, Firefox, Edge...).
